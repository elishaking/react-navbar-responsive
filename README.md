# Responsive Navbar

- For React
- Responsive Navbar component

https://www.npmjs.com/package/react_navbar_responsive

## Install

```cmd
npm install react_navbar_responsive --save
```

## Usage

```javascript
import React from "react";
import "./App.css";
import Navbar from "react_navbar_responsive";

function App() {
  return (
    <div className="App">
      <Navbar />
    </div>
  );
}

export default App;
```
